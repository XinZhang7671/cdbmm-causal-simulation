# CDBMM Causal Simulation
This repository contains the simulation study developed for my master's thesis to evaluate the performance of the Confounder-Dependent Bayesian Mixture Model (CDBMM) for treatment effect estimation and latent subgroup discovery.
## Simulation Workflow
1. Generate synthetic datasets.
2. Estimate treatment effects using CDBMM.
3. Compare performance with BART and CART.
4. Evaluate model performance using Bias, MSE, Relative Error Ratio, and ARI.
